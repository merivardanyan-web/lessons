Դասի նպատակներ

Այս դասից հետո հետևյալն ես սովորելու
  • Ինչ է Flexbox-ը և ինչու է այն CSS-ի ամենակարևոր գործիքը
  • Ինչպես դարձնել ծնողը flex-կոնտեյներ՝ display: flex-ով
  • Ինչպես հավասարեցնել տարրերը հորիզոնально՝ justify-content-ով
  • Ինչպես հավասարեցնել տարրերը ուղղահայաց՝ align-items-ով

Ի՞նչ է Flexbox-ը

CSS-ում տարրերը լռելյայն բաշխվում են բլոկային (block) կամ տողային (inline)։ Բլոկային տարրերը յուրաքանչյուրը իր տողում են, իսկ տողայինը՝ կողք կողքի, բայց չափերով կառավարել չես կարող։

Flexbox-ը (Flexible Box) CSS-ի դասավորության մոդել է, որը թույլ է տալիս ծնողին (parent) վերահսկել, թե որդիները (children) ինչպես են բաշխվում տարածության մեջ։

Ծնողը (Container)

Սա էջի բաժինն է, որին ավելացնում ես display: flex;։ Նա ակտիվացնում է Flexbox մոդելն իր բոլոր երեխաների համար։

Երեխաները (Items)

Կոնտեյների ներսում գտնվող տարրերը։ Լռելյայն իրար կողք են դնվում և կարող են ձգվել կամ սեղմվել՝ ըստ կոնտեյների չափի։

Main Axis

Հիմնական առանցք, որով բաշխվում են տարրերը։ Լռելյայն՝ ձախից աջ (row)։ Փոխում է flex-direction-ով։

justify-content — Հորիզոնական դասավորություն

Այս հատկությունը վերահսկում է, թե ինչպես են տարրերը բաշխվում հիմնական առանցքով (լռելյայն՝ ձախից աջ)։

justify.css
.container { display: flex; justify-content: center; /* կենտրոն */ justify-content: flex-start; /* ձախ (լռելյայն) */ justify-content: flex-end; /* աջ */ justify-content: space-between; /* հավասար բացում */ justify-content: space-around; /* բացում եզրերին */ }

Տեսողական համեմատություն

flex-start (լռելյայն)

center

flex-end

space-between

space-around

align-items — Ուղղահայաց դասավորություն

Այս հատկությունը վերահսկում է, թե ինչպես են տարրերը հավասարվում ընդհանուր առանցքով (ուղղահայաց)։ Աշխատում է միայն այն դեպքում, երբ կոնտեյներն ունի բարձրություն։

align.css
.container { display: flex; height: 200px; align-items: center; /* կենտրոն (ուղղահայաց) */ align-items: flex-start; /* վերև */ align-items: flex-end; /* ներքև */ align-items: stretch; /* ձգում (լռելյայն) */ }

Տեսողական համեմատություն

align-items: flex-start

align-items: center

align-items: flex-end

align-items: stretch (լռելյայն)

flex-direction — Առանցքի ուղղությունը

Փոխում է հիմնական առանցքի ուղղությունը։ Լռելյայն row-ն է՝ ձախից աջ։ column-ով տարրերը դասավորվում են վերևից ներքև։

direction.css
.container { display: flex; flex-direction: row; /* ձախ → աջ (լռելյայն) */ flex-direction: row-reverse; /* աջ → ձախ */ flex-direction: column; /* վերև → ներքև */ flex-direction: column-reverse; /* ներքև → վերև */ }

row

column

Կենտրոնացում՝ երկու առանցքներով

Flexbox-ի ամենահայտնի օգտագործումը՝ երկու առանցքով կենտրոնացում։ Պարզապես 2 հատկություն՝ և ցանկացած բովանդակություն կենտրոնում է։

center.css
/* Կլասիկ՝ բովանդակությունը կենտրոնում */ .hero { display: flex; justify-content: center; /* հորիզոնական */ align-items: center; /* ուղղահայաց */ height: 300px; }
Կենտրոնում
justify-content: center + align-items: center

flex-wrap — Տողափոխություն

Լռելյայն flex տարրերը սեղմվում են մեկ տողում։ wrap-ով ավելորդ տարրերը տեղափոխվում են հաջորդ տող։

wrap.css
.container { display: flex; flex-wrap: nowrap; /* լռելյայն՝ մեկ տող */ flex-wrap: wrap; /* տեղ չլինի՝ նոր տող */ }

flex-wrap: wrap (նեղ պատկերում)

gap — Բացում տարրերի միջև

Փոխարինում է margin-ի օգտագործումը flex-տարրերի միջև։ Ավելի հարմար է, որովհետև զերծում է արտաքին եզրերից՝ ավելացնելով բացում միայն տարրերի միջև։

gap.css
.container { display: flex; gap: 16px; /* բոլոր կողմերով */ gap: 8px 24px; /* ուղղահայաց հորիզոնական */ }

Տարրերի հատկություններ

Մինչև հիմա խոսեցինք կոնտեյների մասին։ Այժմ՝ երեխաները։ Այս հատկությունները գրվում են ոչ թե կոնտեյներին, այլ ինքնուրույն տարրերին։

order

Փոխում է տարրերի հերթականությունը։ Լռելյայն բոլորն ունեն 0։ Փոքր թիվը գալիս է առաջ։

flex-grow

Որքան մեծանա տարրը՝ ազատ տարածքը լցնելու համար։ 1 = բաժանել հավասար, 2 = կրկնակի։

flex-shrink

Որքան սեղմվի տարրը, երբ չափը չի բավարարում։ Լռելյայն 1։ 0 = չսեղմվել։

items.css
.first { order: -1; /* գալիս է ամեն առաջ */ flex-grow: 2; /* կրկնակի տարածք */ } .last { order: 1; /* գալիս է վերջ */ flex-shrink: 0; /* չի սեղմվում */ } .item { align-self: flex-end; /* միայն այս տարրն ավելի ներքև */ }

Խորհուրդներ Flexbox-ի համար

1. Միշտ սկսիր display: flex-ով, հետո ավելացրու մնացածը։ Առանց դրա մնացած հատկությունները չեն աշխատի։
2. justify-content աշխատում է հիմնական առանցքով (row = հորիզոնական, column = ուղղահայաց)։ align-items՝ ընդհանուր առանցքով (հակառակը)։
3. align-items չի աշխատում, եթե կոնտեյներն ունի height: auto (լռելյայն)։ Տուր height կամ min-height։
4. gap-ը ավելի լավ է քան margin-ը flex-տարրերի միջև, որովհետև չի ստեղծում արտաքին ծայրամասեր։
5. Chrome DevTools-ը ցույց է տալիս flex-առանցքը (F12 → Elements → կտտացրիր flex կոնտեյներին → see orange lines)։

Դասի ընթացքում կգրենք

Ստեղծիր 3 card ռяд՝ flexbox-ով և կենտրոնացրու landing page-ի hero բաժինը։

style.css — Պրակտիկա
/* 1. Hero կենտրոնում */ #hero { display: flex; flex-direction: column; justify-content: center; align-items: center; height: 400px; } /* 2. Cards ռяд */ .cards { display: flex; justify-content: space-between; gap: 24px; padding: 40px; } /* 3. Մեկ card */ .card { flex: 1; padding: 24px; border: 1px solid #ddd; border-radius: 8px; }

Տնային աշխատանք — Flexbox Landing Page
Վերցրու քո Landing Page-ը և վերափոխիր այն՝ օգտագործելով Flexbox։
01
#hero-ին տուր display: flex;, flex-direction: column; և justify-content: center;՝ բովանդակությունը ուղղահայաց կենտրոնում դնելու համար։
02
Nav-ին (կամ ul-ին) տուր display: flex;, justify-content: center; և gap: 24px;՝ հղումները կողք կողքի դնելու համար։
03
Ստեղծիր .cards կոնտեյներ՝ display: flex; և justify-content: space-between;։
04
Card-երին տուր flex: 1;՝ հավասարաչափ բաշխման համար, և padding: 24px;։
05
Footer-ին տուր display: flex;, justify-content: space-between; և align-items: center;։
06
Սեփական ոճ. Ավելացրու minimum 2 հատկություն՝ օգտագործելով միայն այս դասում սովորածը (display, flex-direction, flex-wrap, justify-content, align-items, align-self, gap, flex, order, flex-grow, flex-shrink)։
