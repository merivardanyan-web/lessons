Դասի նպատակներ

Այս դասից հետո հետևյալն ես սովորելու
  • Ինչպես օգտագործել HTML-ի հիմնական տեքստային տարրերը
  • Ինչպես ավելացնել նկարներ էջում՝ <img> tag-ով
  • Ինչպես ստեղծել հղումներ՝ ներքին և արտաքին
  • Ինչ է relative path-ը և ինչ է absolute path-ը

Հղումներ՝ <a> tag

<a> tag-ը (anchor) օգտագործվում է մեկ էջից մյուսին անցնելու համար։ Ամենակարևոր attribute-ը href-ն է, որտեղ գրվում է հասցեն։

links.html
<!-- Արտաքին հղում --> <a href="https://google.com">Գնալ Google</a> <!-- Նոր թաբում բացել --> <a href="https://google.com" target="_blank">Բացել նոր պատուհանում</a> <!-- Ներքին հղում --> <a href="about.html">Իմ մասին</a> <!-- Էլ. փոստ --> <a href="mailto:email@example.com">Գրիր ինձ</a>
Հիշիր. target="_blank" ատրիբուտը բացում է հղումը նոր թաբում։ Ներքին հղումների դեպքում սովորաբար պետք չի։

Ինչպես ավելացնել նկարներ

<img>-ը ինքնափակ (self-closing) tag է։ Փակվող ձև չունի։ Պարտադիր src (source) և alt (ալտերնատիվ տեքստ) attribute-ներն են։

images.html
<!-- Տեղական նկար --> <img src="images/photo.jpg" alt="Իմ լուսանկարը" width="300"> <!-- Արտաքին նկար --> <img src="https://site.com/logo.png" alt="Լոգո"> <!-- alt-ը պարտադիր է — էկրանի Vorleser-ների համար --> <img src="banner.jpg" alt="Զեղչեր ամառվա համար">

Relative Path

Ֆայլի հասցեն՝ HTML ֆայլի նկատմամբ։ Օրինակ՝ images/photo.jpg կամ ../logo.png (մեկ թղթապանակ վերև)։

Absolute Path

Լրիվ URL հասցե։ Օրինակ՝ https://example.com/img.jpg։ Չի կախված HTML ֆայլի տեղից։

Այսօր ծանոթանում ենք այս tag-երին

TAG ՆԿԱՐԱԳՐՈՒԹՅՈՒՆ ՕՐԻՆԱԿ
<a> Հղում — anchor, href attribute-ով <a href="url">Տեքստ</a>
<img> Նկար — ինքնափակ, src և alt պարտադիր <img src="a.jpg" alt="">
<ul> + <li> Անկանոն ցուցակ — unordered list <ul><li>...</li></ul>
<ol> + <li> Համարակալված ցուցակ — ordered list <ol><li>...</li></ol>
<pre> Նախֆորմատավորված տեքստ — պահպանում է բացատները <pre>...</pre>
<code> Կոդի հատված — ինլայն կոդ <code>...</code>

Դասի ընթացքում կգրենք

VS Code-ում ստեղծիր portfolio.html ֆայլը և կրկնիր հետևյալ կոդը։

portfolio.html — Պրակտիկա
