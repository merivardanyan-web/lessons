Դասի նպատակներ

Այս դասից հետո հետևյալն ես սովորելու
  • Ինչպես ստեղծել աղյուսակներ՝ <table>, <tr>, <td>, <th>
  • Ինչպես օգտագործել ցուցակներ՝ <ul>, <ol>, <li>, <dl>
  • Ինչպես ստեղծել ֆորմաներ՝ <form>, <input>, <button>
  • Ինչ է label-ը և ինչու է պետք յուրաքանչյուր input-ին

Աղյուսակներ՝ <table>

Աղյուսակները օգտագործվում են տվյալներ կառուցվածքային ձևով ցուցադրելու համար։ Հիմնական tag-երն են՝ <table>, <tr> (տող), <th> (գլխագիր), <td> (վանդակ)։

table.html
<table> <tr> <th>Անուն</th> <th>Տարիք</th> <th>Քաղաք</th> </tr> <tr> <td>Աննա</td> <td>25</td> <td>Երևան</td> </tr> <tr> <td>Արմեն</td> <td>30</td> <td>Գյումրի</td> </tr> </table>
Հիշիր. Աղյուսակներն ավելի լավ է օգտագործել տվյալների համար, ոչ թե էջի դասավորության։ Դասավորության համար հետո կսովորենք CSS-ը։

Ցուցակներ

HTML-ում կան 3 տեսակի ցուցակներ։ ul-ը կետերով է, ol-ը՝ համարակալված, dl-ը՝ սահմանումների ցուցակ։

<ul> — Անկանոն

Կետերով ցուցակ։ Օգտագործվում է, երբ հերթականությունը կարևոր չէ։

<ol> — Համարակալված

Համարներով ցուցակ։ Օգտագործվում է, երբ հերթականությունը կարևոր է։

lists.html
<!-- Անկանոն ցուցակ --> <ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul> <!-- Համարակալված ցուցակ --> <ol> <li>Միացնել համակարգիչը</li> <li>Բացել VS Code-ը</li> <li>Ստեղծել index.html</li> </ol> <!-- Սահմանումների ցուցակ --> <dl> <dt>HTML</dt> <dd>Վեբ էջերի նշագրման լեզու</dd> <dt>CSS</dt> <dd>Վեբ էջերի ոճավորման լեզու</dd> </dl>

Ֆորմաներ՝ <form>

Ֆորմաները օգտագործվում են օգտատերերից տվյալներ ստանալու համար։ Հիմնական tag-երը՝ <form>, <input>, <label>, <button>, <textarea>, <select>։

form.html
<form action="/submit" method="POST"> <label for="name">Անուն</label> <input type="text" id="name" name="name"> <label for="email">Էլ. փոստ</label> <input type="email" id="email" name="email"> <label for="message">Հաղորդագրություն</label> <textarea id="message" name="message"></textarea> <button type="submit">Ուղարկել</button> </form>
Կարևոր. label-ը պարտադիր է ամեն input-ի համար։ Այն օգնում է screen reader-ներին և բջջային օգտատերերին՝ ավելի մեծ տարածք սեղմելու համար։

Դասի ընթացքում կգրենք

VS Code-ում ստեղծիր contact.html ֆայլը և կրկնիր հետևյալ կոդը։

contact.html — Պրակտիկա
<!DOCTYPE html> <html lang="hy"> <head> <meta charset="UTF-8"> <title>Կապ</title> </head> <body> <h1>Կապվիր ինձ հետ</h1> <form> <label for="name">Անուն</label> <input type="text" id="name" name="name"> <label for="email">Էլ. փոստ</label> <input type="email" id="email" name="email"> <button type="submit">Ուղարկել</button> </form> <h2>Իմ սոցիալական ցանցերը</h2> <ul> <li><a href="https://github.com">GitHub</a></li> <li><a href="https://linkedin.com">LinkedIn</a></li> </ul> </body> </html>
